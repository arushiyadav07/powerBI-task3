# Sales Forecasting Using Power BI

## Project Overview
This project demonstrates sales forecasting using historical sales data in Power BI.

## Tools Used
- Power BI Desktop
- Microsoft Excel

## Dataset
The dataset contains:
- Date
- Sales

## Features
- Historical sales trend visualization
- Sales forecasting using Power BI Forecast feature
- Interactive line chart

## Files
- Task3_Sales_Forecasting.pbix
- sales_data.xlsx

## Outcome
This project predicts future sales trends based on historical data using Power BI.
